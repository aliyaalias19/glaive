# Evidence Samples

GLAIVE is tested against public DFIR cases. Raw evidence is **not** committed
to this repository (large files, sometimes redistribution-restricted, and you
should download it yourself to verify integrity).

This directory contains:
- `MANIFEST.yaml` — declarative list of cases, source URLs, and SHA-256 hashes
- One subdirectory per case (created on download)

## Primary case source

SANS provides starter case data at:
https://sansorg.egnyte.com/fl/HhH7crTYT4JK

(Link from the FIND EVIL! Devpost Resources page.)

Final case selection happens on Day 7 (see `docs/CASE_CANDIDATES.md`).
